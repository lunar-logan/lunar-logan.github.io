System design notes

General information

  • Single region can have multiple availability zones
  • Amazon AWS RDS supports cross-region read-replica replication for PostgresSQL
  • This is done vai streaming WAL (transaction logs for PostgreSQL)

MongoDB

  • Supports document level ttl. Expired documents are deleted by a backgroud thread
  • Delete thread runs every 60s, so there could be a delay upto a minute
  • Deleted documents could be streamed via change streams on delete

TinyURL design

  • We could use multiple domains to scale. For eg. tinyurl.com, other.co, etc
